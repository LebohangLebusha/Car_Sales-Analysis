## Bright Motors Car Sales Analysis

## Project Overview

This project analyzes historical car sales data for Bright Motors, where I take on the role of a Junior Data Analyst.

Bright Motors has recently appointed a new Head of Sales, whose mission is to:

* Expand the dealership network
* Improve sales performance
* Optimize inventory

This analysis focuses on extracting actionable insights to support strategic decision-making for future sales and marketing efforts.

---

## Business Objectives

The project aims to answer the following key questions:

* Which car makes and models generate the most revenue
* The relationship between price, mileage, and year of manufacture
* Which regions or locations have the highest sales volumes
* Emerging trends in customer purchasing preferences
* Recommendations to increase profitability and operational efficiency

---

## Analytical Focus

In addition to the core objectives, this analysis explores:

* Which cars have high future sales potential based on historical trends
* Customer preference: Manual vs Automatic transmission
* Which region/state generates the highest revenue
* Which body type, make, and model drive the most revenue
* Performance of car manufacturing years (Top 10 best-performing years)
* Which dealership generates the most revenue
* Relationship between car year, mileage, and selling price

---

## Dataset Overview

The dataset contains detailed vehicle sales information with the following columns:

| Column Name  | Description                                    |
| ------------ | ---------------------------------------------- |
| year         | Manufacturing year of the vehicle              |
| make         | Brand of the car (e.g., Toyota, BMW)           |
| model        | Specific model of the car                      |
| trim         | Variant or version of the model                |
| body         | Body type (e.g., SUV, Sedan, Hatchback)        |
| transmission | Type of transmission (Manual / Automatic)      |
| vin          | Unique Vehicle Identification Number           |
| state        | Location where the car was sold                |
| condition    | Condition rating of the vehicle                |
| odometer     | Mileage of the vehicle                         |
| color        | Exterior color                                 |
| interior     | Interior color/type                            |
| seller       | Dealer or seller name                          |
| mmr          | Market valuation (Manheim Market Report value) |
| sellingprice | Final selling price of the vehicle             |
| saledate     | Date when the car was sold                     |

## Project Workflow

### 1. Planning and Design

* Created a mind map in Miro to structure the analysis
* Built a Gantt chart in Canva to manage project timelines

### 2. Data Preparation

* Converted raw dataset into a usable format
* Cleaned data (handled missing values, duplicates, formatting issues)

### 3. Data Processing (Databricks)

* Aggregated data using SQL
* Created calculated fields:

  * Total Revenue = Selling Price × Units Sold
  * Profit Margin = (Selling Price - Cost Price) / Selling Price × 100

### 4. Data Analysis (Excel)

* Used pivot tables to explore trends
* Built charts to analyze performance across multiple dimensions

### 5. Dashboard Development

* Developed interactive dashboards in Google Data Studio
* Included filters for:

  * Region
  * Fuel Type
  * Year
  * Transmission

### 6. Insights and Reporting

* Identified key business insights
* Developed strategic recommendations
* Prepared presentation for stakeholders
